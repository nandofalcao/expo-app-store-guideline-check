## 8. Segurança dos Dados

Adotamos medidas técnicas e administrativas para proteger seus dados:

- **Encriptação em trânsito:** TLS 1.3 em todas as comunicações
- **Encriptação em repouso:** dados sensíveis armazenados de forma encriptada
- **Acesso restrito:** somente colaboradores autorizados acessam dados pessoais
- **Monitoramento:** sistemas de detecção de acesso não autorizado
- **Revisões regulares:** auditorias periódicas de segurança

Em caso de incidente de segurança que possa afetar seus dados, notificaremos você e a Autoridade Nacional de Proteção de Dados (ANPD) conforme exigido pela LGPD.

## 9. Crianças e Adolescentes

[ESCOLHA UMA DAS OPÇÕES ABAIXO:]

**Opção A — App NÃO é para menores:**
Este Aplicativo não é destinado a pessoas menores de 18 anos. Não coletamos intencionalmente dados de menores. Se descobrirmos que coletamos dados de uma criança sem consentimento parental, excluiremos essas informações imediatamente.

**Opção B — App é para menores:**
Para usuários menores de 12 anos, exigimos o consentimento expresso de um responsável legal. Coletamos apenas os dados mínimos necessários para o funcionamento do serviço e não compartilhamos dados de crianças com terceiros para fins de publicidade.
